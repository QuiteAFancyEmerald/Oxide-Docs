## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- yarn
  ```sh
  npm install yarn@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Flow-Works/FlowOS
   ```
2. Install packages
   ```sh
   yarn install
   ```
3. Run
   ```sh
   yarn start
   ```
   or for Windows
   ```powershell
   yarn win
   ```
